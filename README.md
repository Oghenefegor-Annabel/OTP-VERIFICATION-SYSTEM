# OTP-VERIFICATION-SYSTEM
This is my final project for Code In Place, 6 weeks Plus lecture on coding
## Description

This project is a Python-based OTP (One-Time Password) verification application developed as a final project for Code in Place. This application generates secure OTP codes and verifies user input with limited attempts and expiration timing.

## Features

* Random OTP generation
* Alphanumeric OTP support
* OTP expiration timer
* Verification system
* Limited login attempts
* Console-based interface

## Technologies Used

* Python
* Random Module
* String Module
* Time Module

## How to Run

1. Install Python
2. Open terminal in project folder
3. Run:

python main.py
